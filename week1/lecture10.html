Lecture 10: Displaying Images

  1. Displaying an image with src attribute

    <img src="../img/img_lights.jpg">

  2. Displaying an image with src,width and height attributes

    <img src="../img/img_lights.jpg" width="200" height="200">

  3. If a path or an url are incorrect, the image will not be displayed

    <img src="../img/img_lghts.jpg" width="200" height="200" alt=.."invalid image url">
